# Slack → Element

## 软件信息
- 商业软件：Slack
- 开源替代：Element
- 厂商：Salesforce
- 许可证：AGPL-3.0（基于 Matrix 协议）

## 功能对比
| 功能 | Slack | Element |
|------|-------|---------|
| 频道 | ✅ | ✅ |
| 私信 | ✅ | ✅ |
| 话题回复 | ✅ | ✅ |
| 端到端加密 | ⚠️ 仅企业版 | ✅ 默认开启 |
| 自托管 | ❌ | ✅ |
| 音视频通话 | ✅ | ✅ |
| 机器人与集成 | ✅ | ✅ |

## 优势
- 去中心化协议（Matrix）
- 默认端到端加密
- 可自托管，数据完全自主
- 跨服务器联邦通信

## 劣势
- 用户体验不如 Slack 精致
- 集成市场较小
- 自托管需承担服务器运维

## 迁移难度
**中等** —— 频道结构可直接映射，但 Slack 专属应用和工作流需替换为 Matrix 机器人和桥接。用户需适应联邦身份模型。
