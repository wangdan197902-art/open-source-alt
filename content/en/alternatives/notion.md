# Notion → Obsidian

## Software Information
- Commercial software: Notion
- Open source alternative: Obsidian
- Vendor: Notion Labs
- License: Proprietary core / Open plugin ecosystem

## Feature Comparison
| Feature | Notion | Obsidian |
|---------|--------|----------|
| Markdown notes | ✅ | ✅ |
| Database / tables | ✅ | ⚠️ Via plugins |
| Real-time collaboration | ✅ | ⚠️ Via Sync |
| Offline access | ⚠️ Limited | ✅ |
| Local storage | ❌ | ✅ |
| Bidirectional links | ✅ | ✅ |

## Advantages
- Local-first storage, full control of your data
- Rich plugin ecosystem
- Fast, offline-ready
- Bidirectional linking for knowledge graphs

## Disadvantages
- Collaboration features require paid sync
- Steeper learning curve for non-technical users
- Database features less powerful than Notion

## Migration Difficulty
**Medium-High** — Note structures can be exported as Markdown, but Notion databases and views need to be reorganized as Obsidian properties or plugin-based tables.
